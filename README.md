# 云麦体重数据同步到佳明

这个项目可以将云麦好轻智能体脂秤的数据同步到佳明（Garmin Connect）账号。支持同步体重、体脂率、水分率、肌肉量等多项身体指标数据。

## 功能特点

- 支持云麦体重数据自动同步到佳明
- 支持佳明中国区和国际区账号
- 支持指定时间范围的数据同步
- 支持Token缓存，避免频繁登录
- 同步多项身体指标数据：
  - 体重
  - 体脂率
  - 水分率
  - 肌肉量
  - 骨量
  - 基础代谢
  - 蛋白质
  - 体年龄
  - 内脏脂肪等级
  - BMI
